Задание состоит из двух частей.

1 часть.
Можно использовать любые языки программирования, их комбинацию, любые библиотеки,
любые приёмы программирования. Можно даже попросить нейросеть написать код.

Собственно, задание:
Создать тестирующую систему, которая проверяет выход ./<program_name>.exe,
то есть проверяет, что печатает в терминал другая программа при определенных входных данных.
Нужно проверить, будет ли первый стек отсортирован после применения
напечатаннпой последовательности команд и наличие всех чисел последовательности.
Другими словами, нужно подготовить тесты для программы из второй части.

2 часть.
Создать программу на языке Си++, которая выводит в терминал последовательность команд для
сортировки стека.
Правила "игры":
Всего есть три стека. В первом лежат случайные числа, не более 10 чисел. Второй и третий стеки пустые.
Разрешено только перекладывать по одному числу за раз из одного стека в другой, при чём
нельзя класть большее число на меньшее. Необходимо, чтобы после всех перекладываний
первый стек был отсортирован по убыванию и в нём были все числа, которые были в нём сначала.

Не смотря на формулировку задачи, программа решающая задачу не обязана иметь 3 стека.
В ней вообще может не быть стеков. Разрешены любые структуры данных. Рекурсия запрещена.

На вход поступает последовательность чисел: содержимое первого стека.

На линукс тоже должно работать!

Считывание с консоли целые положительные и отрицательные:

Вывод в stdout:

1 (2) 3

3 (2) 1
2 (1) 1
